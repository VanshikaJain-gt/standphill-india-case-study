# 🌐 Standphill India Website Redesign & Development Case Study

> Complete frontend redesign and implementation of Standphill India's live business website during my internship as a Frontend Developer & UI/UX Designer.

🔗 **Live Website:** https://www.standphillindia.in

---

## 📋 Project Overview

During my internship at Standphill India, I was given the responsibility of redesigning and rebuilding the company's website frontend to create a more modern, responsive, and user-friendly experience.

The project involved transforming the existing website into a professional business platform that effectively showcases the company's services while improving usability, accessibility, and mobile responsiveness.

---

## 👩‍💻 My Role

**Frontend Developer & UI/UX Designer**

### Responsibilities

* Complete UI/UX redesign from scratch
* Frontend development and implementation
* Responsive design for desktop, tablet, and mobile devices
* Navigation structure redesign
* Interactive component development
* Contact and consultation form integration
* WhatsApp CTA integration
* Performance optimization
* Cross-browser testing
* Collaboration with backend and deployment teams
